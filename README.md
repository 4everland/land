# Land

contracts for land

## Local development

create `.env` file and set your mnemonic as below

   `MNEMONIC="your mnemonic"`

`npm i`

`npm run build`

`npx hardhat node --no-deploy`

`npx hardhat deploy --tags ${deploymentTag}`

#### notice
Once the rpc url is unavailable, check it [here](https://chainlist.org/)
